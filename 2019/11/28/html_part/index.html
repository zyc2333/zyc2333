  • 超文本标记语言，不是编程语言。

1.文档声明类型：<!DOCTYPE HTML>不属于html

W3C出来之前，不同的浏览器有不同的渲染方式，这种方式叫混杂模式，W3C标准出来以后有了统一的标准模式,若文档类型声明不正确，会导致浏览器以混杂模式渲染。
