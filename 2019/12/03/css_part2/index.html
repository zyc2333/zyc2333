flex布局（Flexible Box)

  • 简介

    弹性盒布局，是一种一维布局模型。

  • 组成

    container\item\main axis\cross axis

  • 例子：

flex-direction

  • 规定 flex items 排列的方向，设置的方向即是主轴 main cross 的方向

  • 参数：

    row(行) : 默认值，子项水平排列

    column（列）: 子项从上到下排列

    row-reverse: 子项水平逆序排列

    column-reverse: 子项垂直逆序排列

  • 例子：

flex-wrap

  • 当子项是固定宽度且超出容器时是否折行展示

  • 参数：

    no-wrap: 默认值，所有子项放置到一行

    wrap: 子项会折行(从上到下折)

    wrap-reverse: 子项会折行(从下到上折)

  • 例子：

justify-content（不是水平居中）

  • 规定子项沿着主轴的分布方式
  • 例：width/height必须写

align-items

  • 规定子项沿着纵轴的分布方式
  • 例：

align-content

  • 如何沿着纵轴在内容项之间和周围分配空间

实现水平居中

子项属性：order

子项占用空间

  • 若子项尺寸累加小于容器空间，子项会按比例延展充满容器剩余空间
  • 若子项尺寸累加大于容器空间，子项会按比例收缩直到充满容器空间

子项属性：flex-basis

  • 对于子项1，flex-basis 未设置默认是auto，也未设置width，子项基准尺寸由内容决定。
  • 对于子项2，flex-basis 为auto，子项占用宽度使用width的宽度，所以子项基准尺寸是70px。
  • 对于子项3，flex-basis 不为auto，子项基准尺寸使用flex-basis设置的值100px。

子项属性：flex-grow

  • 设置子项按何种比例“瓜分”容器剩余空间
  • 默认值是0
  • 子项实际尺寸 = 该子项基准尺寸 + (容器空间 - 所有子项基准尺寸之和) * 该子项 flex-grow / 所有子项 flex-grow 之和

子项属性：flex-shrink

  • 设置子项按何种比例“吸收”超出容器的空间
  • 默认为1
  • 子项实际尺寸 = 该子项基准尺寸 - (所有子项基准尺寸之和 - 容器空间 ) * (该子项基准尺寸 * flex-shrink ) / 所有子项基准尺寸 * flex-shrink之和

子项属性：flex

  • flex-grow、flex-shrink、flex-basis的结合

  • EXAMPLE：

  • flex: auto 和 flex: 1有什么区别

    flex: 1:
    flex-basis: 0; flex-grow: 1; flex-shrink: 1;
    容器所有的空间都作为剩余空间来平分
    最终效果是子项完全等分

    flex: auto
    flex-basis: auto; flex-grow: 1; flex-shrink: 1;
    容器先去除子项内容占据的空间，再平分

  • flex实现两栏布局

grid布局

  • 是一种二维布局模型，先在页面上画好“虚拟格子”，再设置元素在格子上的位置和跨度

容器：设置grid布局

display: grid; /*inline-grid*/

容器：画虚拟格子

grid: 30px auto 30px / 10% 10px 1fr 10px 10%; (3行5列)
  • grid：行高/列宽
  • auto一般用于设置行高时，fr表示占据剩余空间，用于设置列宽时

子项：放置自己的位置

grid-area: 3/3/4/5;
  • grid-area:起始横线/起始竖线/终止横线/终止竖线

浮动

  • 不推荐布局使用
  •   container::after {
  content: '';
  display: block;
  clear: both;
  }
  • 浮动元素半脱离普通流，后续的块级元素发现不了它，后续的行级元素围绕它

BFC(Block Formatting Context)

  • 特性

    属于同一个BFC的两个相邻盒子的外边距会发生折叠。
    BFC的区域不会与浮动盒重叠。
    创建BFC就是页面上的一个隔离的独立容器，容器里面的子元素不会影响到外面的元素。反之也如此。
    计算BFC的高度时，浮动元素也参与计算。

  • 形成条件

    • 根元素html
    • float属性不为none；
    • position为absolute、fixed
    • display为inline-block、table-cell
    • display: flow-root 或者 flow-root list-item
    • overflow为hidden、scroll、auto；
    • flex和grid的子项

  • 作用

    • 包含内部浮动

    • 排除外部浮动

    • 阻止外边距折叠

      阻止父子折叠，不能阻止相邻折叠
